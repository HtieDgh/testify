# Changelog

## [Unreleased] 

## [v0.2.1] - 2024-12-24

### Added

- Возможность удаления теста одновремено с его вариантами на странице пользователя
- Возможность удаления отдельных вариантов теста на странице редактора теста

## [v0.2.0]  - 2024-12-24

### Added

- Редактор, который позволяет добавлять/изменять тесты
- Пользователь с ролью "Создатель теста" и выше теперь может добавлять/изменять свои тесты
- Возможность создавать/изменять Варианты теста
- Возможность создавать/изменять Вопросы и ответы для них
- Возможность прикреплять файлы
- Загрузка резервной копии теста по окончанию редактирования


### Changed

- Профиль на странице отображает созданые тесты у "Создалтеля тестов" с кнопками управления: изменить тест, добавить тест.

## [v0.1.1] - 2024-12-15

### Added

- Регистрация новго пользователя на странице /regist
- Зарегистрироваться с помощью интерфейса можно только как Участник теста 

### Changed

- Профиль теперь учитывает уровень доступа пользователя с ролью Создатель и Участник теста.

## [v0.1.0] - 2024-12-14

### Added

- Добавлена возможность авторизации пользователя и входа в профиль.
