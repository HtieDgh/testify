# Changelog

## [v0.4.1] - 2024-12-26

## Added

- Руководство пользователя

## Fixed

- мелкие ошибки и недочеты

## [v0.4.0] - 2024-12-25

## Added

- UI Менеджера
- Изменение учетных записей: Добавлена страница с формой редактирования учетных записей для Менеджера
- Просмотр статистики любого теста у Менеджера

## [v0.3.1] - 2024-12-25

### Fixed

- При попытке пройти тест теперь проверяется его период проведения и если он окончен товыводится сообщение
- При вставке нового вопроса в изменяемый вариант интерфейс обрабатывает его коректно
- Вывод попыток на странице профиля

## [v0.3.0] 

## Added

- Плеер для теста: Теперь тесты можно прходить, сохраняя попытки в системе.
- Возможность просмотра статистики у Создателя теста у конкретного теста, с учетом вариантов и возможностью поиска по попыткам и вариантам
- Возможность просмотра попыток для всех ролей пользователей
- Поиск по тестам и попыткам на странице профиля и на странице со статистикой по конкретному тесту

### Changed

- Теперь отображается кол-во вопросов на странице редактора теста, для последующего удобного редактировнаия
- На странице с профилем отображаются попытки пользователя

### Fixed

- Завершение редактирования теста теперь переводит на главный экран
- Мелкие ошибки при сохранении вопросов/вариантов

## [v0.2.1] - 2024-12-24

### Added

- Возможность удаления теста одновремено с его вариантами на странице пользователя
- Возможность удаления отдельных вариантов теста на странице редактора теста

## [v0.2.0]  - 2024-12-24

### Added

- Редактор, который позволяет добавлять/изменять тесты
- Пользователь с ролью "Создатель теста" и выше теперь может добавлять/изменять свои тесты
- Возможность создавать/изменять Варианты теста
- Возможность создавать/изменять Вопросы и ответы для них
- Возможность прикреплять файлы
- Загрузка резервной копии теста по окончанию редактирования


### Changed

- Профиль на странице отображает созданые тесты у "Создалтеля тестов" с кнопками управления: изменить тест, добавить тест.

## [v0.1.1] - 2024-12-15

### Added

- Регистрация новго пользователя на странице /regist
- Зарегистрироваться с помощью интерфейса можно только как Участник теста 

### Changed

- Профиль теперь учитывает уровень доступа пользователя с ролью Создатель и Участник теста.

## [v0.1.0] - 2024-12-14

### Added

- Добавлена возможность авторизации пользователя и входа в профиль.
